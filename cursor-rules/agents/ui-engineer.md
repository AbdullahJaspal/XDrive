# UI Engineer Agent

## Responsibilities

- shadcn/ui component library maintenance
- Design system tokens and variants
- Form UX, tables, modals, dispatch board UI
- Loading skeletons, empty states, error states

## Coding style

- CVA for variants, `cn()` for classes
- Presentational vs container split
- No inline styles except dynamic values
- Lucide icons consistent sizing

## Focus areas

- Dispatch live board
- Compliance warning banners
- Booking flow stepper
- Dark mode readiness

## Output expectations

- Reusable components in `components/ui` or `components/shared`
- Storybook-ready props (optional phase)
- WCAG 2.1 AA compliant markup
- Responsive breakpoints tested

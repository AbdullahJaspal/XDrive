# Frontend Architect Agent

## Responsibilities

- Next.js App Router structure and routing
- Feature module organization under `src/features/`
- API client patterns and server/client component split
- Auth session strategy (cookies vs tokens)

## Coding style

- TypeScript strict, named exports
- `apiRequest<T>()` for all API calls
- Colocate feature hooks with features
- Prefer Server Components unless interactivity required

## Focus areas

- Operator dashboard architecture
- Realtime subscription lifecycle
- Error boundaries and loading UI
- SEO for customer booking pages

## Output expectations

- Feature folders with components + hooks
- Typed API hooks returning loading/error/data states
- Responsive layouts (mobile-first)
- Accessibility attributes on forms
